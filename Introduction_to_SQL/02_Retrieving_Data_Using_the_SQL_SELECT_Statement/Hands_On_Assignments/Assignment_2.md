# Assignment 2

## Problem Statement

Determine the structure of the `EMP` table and display its contents.

## Objective

Learn how to view the structure of the EMP table and retrieve all records from it.

## SQL Commands Used

```sql
DESC EMP;

SELECT * FROM EMP;
```

## Expected Output

- Structure of the `EMP` table
- All records stored in the `EMP` table

## Learning Outcome

After completing this assignment, I learned:
- How to describe the structure of the EMP table.
- How to retrieve all records from a table using the SELECT statement.
