# Assignment 1

## Problem Statement

Determine the structure of the `DEPT` table and display its contents.

## Objective

Learn how to view a table structure and retrieve all records from a table.

## SQL Commands Used

```sql
DESC DEPT;

SELECT * FROM DEPT;
```

## Expected Output

- Structure of the `DEPT` table
- All records stored in the `DEPT` table

## Learning Outcome

After completing this assignment, I learned:
- How to describe the structure of a table.
- How to retrieve all records using the `SELECT` statement.
