# Topic 05 - Reporting Aggregated Data Using Group Functions

## Objective

Learn how to summarize and analyze data using Oracle SQL Group Functions and Group By clauses.

---

## Topics Covered

- COUNT()
- SUM()
- AVG()
- MIN()
- MAX()
- ROUND()
- GROUP BY Clause
- HAVING Clause
- Aggregate Functions
- Nested Group Functions

---

## Repository Structure

Each topic contains:

- Hands-on Assignments
- SQL Scripts
- Documentation

---

## Learning Outcome

After completing this topic, you will be able to generate summarized reports, perform calculations on groups of data, and filter grouped records using Oracle SQL.
