# Introduction to SQL

## Module Overview

This module covers the fundamentals of SQL used to retrieve, filter, sort, and manipulate data in relational databases.

## Topics

1. Introduction to SQL
2. Retrieving Data Using the SQL SELECT Statement
3. Restricting and Sorting Data
4. Using Single-Row Functions to Customize the Output
5. Reporting Aggregated Data Using the Group Functions
6. Using the Set Operators
7. Displaying Data from Multiple Tables Using Joins

- README.md
- Hands_On_Assignments
- SQL_Files

After completing this module, I learned how to retrieve, filter, sort, and combine data using SQL.
