# Topic 07 - Displaying Data from Multiple Tables Using Joins

## Objective

Learn how to retrieve related data from multiple tables using different types of SQL joins in Oracle.

---

## Topics Covered

- Natural Join
- Equi Join
- Self Join
- Outer Join
- Non-Equi Join
- USING Clause
- ON Clause
- LEFT OUTER JOIN
- RIGHT OUTER JOIN
- FULL OUTER JOIN

---

## Repository Structure

This topic contains:

- Hands-on Assignments
- SQL Scripts
- Documentation

---

## Learning Outcome

After completing this topic, you will be able to retrieve and combine data from multiple tables using various Oracle SQL join techniques.
