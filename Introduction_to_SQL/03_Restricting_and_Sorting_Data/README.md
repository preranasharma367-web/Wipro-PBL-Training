# Topic 03 - Restricting and Sorting Data

## Objective

Learn how to retrieve specific data from database tables by filtering, sorting, and displaying records using SQL clauses and operators.

## Topics Covered

- WHERE Clause
- ORDER BY Clause
- DISTINCT
- Column Aliases
- Arithmetic Operators
- Comparison Operators
- Logical Operators
- BETWEEN
- IN
- LIKE
- IS NULL
- SQL*Plus Prompts
